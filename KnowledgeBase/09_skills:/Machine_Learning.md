## Machine Learning Projects

### Customer Churn Prediction (PySpark & Google Cloud Platform)

Objective:
Predict customers likely to leave a subscription-based business.

Models Developed:

* Logistic Regression
* Decision Tree
* Random Forest

Key Results:

* Decision Tree achieved the highest Accuracy and F1 Score
* Random Forest achieved the strongest ROC-AUC (0.7407)
* Model performance improved with increased data volume

Technologies:

* PySpark
* Google Cloud Dataproc
* Machine Learning Pipelines
* GCP

### Retail Customer Behavior Analysis

Objective:
Identify high-value customers and improve customer retention strategies.

Techniques:

* RFM Analysis
* K-Means Clustering
* Customer Segmentation
* Predictive Modeling

Results:

* Identified High-Value, Loyal, Regular, and At-Risk customers
* Random Forest achieved 82.9% Accuracy
* ROC-AUC of 0.865

### Olist Brazilian E-Commerce Analytics

Objective:
Analyze customer satisfaction and forecast future sales.

Techniques:

* Customer Segmentation
* Clustering Analysis
* Time Series Forecasting
* ARIMA Modeling

Results:

* Identified key factors influencing customer satisfaction
* Holt's Linear Trend achieved best forecasting performance
* Forecast accuracy improved significantly compared to baseline models
