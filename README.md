# 🤖 Ask Anand AI

### AI-Powered Career & Portfolio Assistant

Ask Anand AI is a Retrieval-Augmented Generation (RAG) based portfolio assistant that enables recruiters, hiring managers, and professionals to interact with my professional profile using natural language.

The application provides instant answers about my experience, analytics projects, leadership achievements, domain expertise, and career journey while also supporting Job Description matching and skill-gap analysis.

## 🏗️ Architecture

```text
User Question
      │
      ▼
Sentence Transformer
(all-MiniLM-L6-v2)
      │
      ▼
ChromaDB Vector Store
      │
      ▼
Semantic Retrieval
      │
      ▼
Relevant Knowledge Chunk
      │
      ▼
Generated Response
```
